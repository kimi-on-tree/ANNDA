This project involved implementing a single-layer perceptron in Python to perform binary classification tasks using both classical perceptron learning and the Widrow-Hoff (delta) rule. The study analyzed how various factors—including learning rates, sequential versus batch training, and the presence of a bias term—influence convergence behavior and the resulting decision boundaries. Findings demonstrated that while both algorithms solve linearly separable tasks, the delta rule provides more stable and consistent solutions, though its generalization is significantly impacted by sampling bias.
